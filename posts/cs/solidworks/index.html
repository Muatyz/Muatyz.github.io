校园授权版? 盗版?#

事实上教育版的时限早在 2022-12-29 就结束了, 我不想让电脑上装一个时不时需要接入校园网才能验证的累赘.

并且根据水源社区上的经验贴:

如果有人要长期使用SolidWorks，请避开交大的学生版，之前发现的问题有：

  • 教育版水印
  • 需要接入校园网验证许可证
  • 好像是建模还是仿真有零件/复杂度上限

所以为什么还要坚持用学生版呢? 直接用盗版一步到位.

盗版资源寻找#

首先要能够理解 Solidworks 的版本号是什么意思.

SolidWorks.2022.SP5.0.Premium

然后直接复制+破解版搜索, 一般都会找到. 最新版本的一般很难找到或者只能是SP1.0, 如果要高一点就只能找更早一年的资源了.

然后是漫长的等待百度网盘挤牙膏的时间.

安装激活步骤#

内附的readme.txt翻译:

readme
  1. Before installation, block the outgoing Internet access by means of Windows Firewall or cord plug. Check .NET Framework 3.5 and 4.0 are installed. If .NET Framework 3.5 (including 2.0) is not installed, go to "Control Panel" -> "Programs and Features" -> "Turn Windows features on or off" ->-> select ".NET Framework 3.5 (including 2.0)"

安装前, 通过 Windows 阻止上网. 如果未安装.NET Framework 3.5(包括2.0), 应当通过

“控制面板”->“程序和功能”->“打开或关闭Windows 功能”-> 选择“.NET Framework 3.5（包括2.0）”

  1. Uninstall (if exist) SolidWorks_Flexnet_Server from SW2020 SSQ's release! To do it run as Administrator. SolidWorks_Flexnet_Server\server_remove.bat and wait until service SolidWorks Flexnet Server will be removed. After that delete SolidWorks_Flexnet_Server folder from computer

卸载SolidWorks_Flexnet_Server文件夹, 通过管理员权限运行SolidWorks_Flexnet_Server\server_remove.bat并等待SolidWorks Flexnet Server服务被删除. 然后删除SolidWorks_Flexnet_Server文件夹.

  1. Run sw2022_network_serials_licensing.reg and confirm to add info into Windows Registry

运行sw2022_network_serials_licensing.reg并确认添加信息到 Windows 注册表.

  1. Copy folder SolidWorks_Flexnet_Server to C: , run as Administrator SolidWorks_Flexnet_Server\server_install.bat and wait until new service SolidWorks Flexnet Server will be install and started

复制SolidWorks_Flexnet_Server文件夹到C:盘, 通过管理员权限运行SolidWorks_Flexnet_Server\server_install.bat并等待SolidWorks Flexnet Server服务被安装并启动.

  1. Install SolidWorks 2022 (including PDM Client if required). DO NOT install SolidNetwork License Server (SNL)! When asked of License Server definition input: 25734@localhost

3.1 If the System Check Warning window appears, ignore it (click Next to continue)

3.2 If the warning "SolidWorks Serial number was not found in activation database" appears, ignore it (click OK to continue)

3.3 If the full list of SW products to install is not visible, click "Select different package" and tick option "Select products not included in this package"

3.4 Select SW products to be installed

3.5 If the "The Installation Manager was unable to determine the current subscription expiration date. Would you like to reactivate your license to update this information?" appears, press No and press Yes in "Do you want to do it later?" prompt.

安装 SolidWorks 2022 (包括 PDM 客户端). 不要安装 SolidNetwork License Server (SNL)! 当询问 License Server 时输入: 25734@localhost

3.1 如果出现系统检查警告窗口，请忽略（点击下一步继续）

3.2 如果出现“SolidWorks Serial number not found in activation database”警告出现，忽略它（点击确定继续）

3.3 如果看不到要安装的软件产品的完整列表，单击“选择不同的包”并勾选选项“选择此包装中未包含的产品”

3.4 选择要安装的软件产品

3.5 如果“安装管理器无法确定当前订阅截止日期。您想重新激活您的许可证以更新此信息吗？”出现时，按否，然后在“您想稍后再做吗？”中按是。迅速的。
  1. After end of setup overwrite original SolidWorks 2020 program folders (if exist) with cracked ones from folder Program Files\SOLIDWORKS Corp and Program Files (x86)

Folders to be replaced from Program Files\SOLIDWORKS Corp (at setup by default):

C:\Program Files\SOLIDWORKS Corp\eDrawings

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS CAM

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS Composer

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS Electrical

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS Explorer

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS Flow Simulation

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS Inspection

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS Manage Client

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS PCB

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS PDM

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS Plastics

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS Visualize

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS Visualize Boost

Folders to be replaced from Program Files (x86) (at setup by default):

C:\Program Files (x86)\SOLIDWORKS PDM

  1. 安装完成后，用破解版文件夹 Program Files\SOLIDWORKS Corp 和 Program Files (x86) 覆盖原始的 SolidWorks 2020 程序文件夹（如果存在）。

要从 Program Files\SOLIDWORKS Corp 替换的文件夹（在默认安装时）包括：

C:\Program Files\SOLIDWORKS Corp\eDrawings

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS CAM

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS Composer

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS Electrical

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS Explorer

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS Flow Simulation

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS Inspection

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS Manage Client

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS PCB

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS PDM

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS Plastics

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS Visualize

C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS Visualize Boost

要从 Program Files (x86) 替换的文件夹（在默认安装时）包括：

C:\Program Files (x86)\SOLIDWORKS PDM

  1. Run SolidSQUADLoaderEnabler.reg and confirm to add info into Windows Registry

运行SolidSQUADLoaderEnabler.reg并确认添加信息到 Windows 注册表.

  1. REBOOT COMPUTER!
  1. 重启电脑!
  1. Run SolidWorks > Help > SolidNetWork License Manager > License Order

Use the "Move Up" and "Move Down" buttons to position Premium products with the same name higher than Professional and Standard products

Click "Apply" to save the settings

Click "OK" to close the SolidNetWork License Manager

运行SolidWorks > 帮助 > SolidNetWork License Manager > License Order

使用“上移”和“下移”按钮将高级产品与相同名称的专业和标准产品放置在更高的位置

单击“应用”以保存设置

单击“确定”以关闭 SolidNetWork License Manager

实际上最后还是选择了使用校园版. 因为盗版的破解流程不知道为什么总是出现问题, 并且反复的安装和卸载也是一个非常麻烦的过程.

为了节约精力最后还是选择了在线授权版, 附带的安装说明书非常详尽.
